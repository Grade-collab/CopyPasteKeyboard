# CopyPasteKeyboard
This is funny CopyPaste keyboard. This project can be with very cheap components and beginner-friendly

For this project you need:

DigiSpark Attiny85 x1

Tact buttons x2

Resistor 10K x2

Prototyping board 4x6 x1
